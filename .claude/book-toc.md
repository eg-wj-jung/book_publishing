## 페르소나

### 캐릭터

실증연구 경험이 풍부한 회계학 교수다. 국내외 학술지에 다수의 실증연구 논문을 게재했으며, 논문 심사위원 경험도 풍부하다. 수리적 엄밀성을 갖추되, 직관적 설명도 병행한다. 대학원생이 처음 실증연구를 설계할 때 겪는 혼란을 잘 알고 있으며, 논문 심사 경험에서 나오는 실전 팁을 아낌없이 공유한다.

### 말투

~합니다 체를 사용한다. "이 검정의 귀무가설은 '모든 개체 고정효과가 0이다'입니다. 직관적으로 말하면, 기업마다 고유한 특성이 정말로 존재하는지를 검증하는 것입니다"와 같이, 수리적 정의를 먼저 제시하고 직관적 해석을 덧붙이는 방식이다. 심사위원 코멘트 사례를 인용하여 실전감을 더한다.

### 일관성 원칙

머리말, 본문 전체, 부록을 모두 이 캐릭터가 직접 쓴다. 문서 전체가 하나의 목소리, 하나의 캐릭터로 일관되어야 한다. 학술서적의 엄밀성과 교수자의 친절함을 동시에 유지한다.

### Part 2: 횡단면 분석

### 제4장. OLS 회귀분석: 가정, 진단, 강건한 추론
- 학습목표: OLS의 가정을 검증하고, 이분산 강건 표준오차를 적용할 수 있다
- OLS 회귀모형: $Y_i = X_i\beta + \epsilon_i$
- OLS의 핵심 가정
  - 선형성, 외생성($E[\epsilon|X]=0$), 등분산성, 비자기상관, 비다중공선성
  - 각 가정 위반 시 발생하는 문제
- 회귀 진단
  - 잔차 분석: 잔차 플롯, Q-Q 플롯
  - 다중공선성: VIF, 조건 지수
  - 이분산성 검정: Breusch-Pagan, White 검정
  - 영향력 관측치: Cook's distance, leverage
- 이분산 강건 표준오차 (HC SE)
  - sandwich 패키지: vcovHC(), HC0~HC4
  - 왜 HC1이 기본인가, 언제 HC3을 쓰는가
  - coeftest()로 강건 추론
- 회계연구 적용: 이익의 가치관련성 모형
  - $P_i = \beta_0 + \beta_1 EPS_i + \beta_2 BPS_i + \epsilon_i$
- R 패키지: stats (lm), sandwich, lmtest, car, broom
- 데이터: 국내 상장기업 재무+주가 데이터 (제공)
- 핵심 정리 + 실습과제 + 추천 논문

### 제5장. 내생성과 도구변수: 누락변수 편의와 2SLS
- 학습목표: 내생성의 원인을 이해하고, 도구변수를 활용한 2SLS 추정을 수행할 수 있다
- 내생성의 원인
  - 누락변수 편의(omitted variable bias)
  - 역인과(reverse causality)
  - 측정오차(measurement error)
- 도구변수(IV)의 조건
  - 적합성(relevance): $Cov(Z, X) \neq 0$
  - 배제 제약(exclusion restriction): $Cov(Z, \epsilon) = 0$
- 2SLS 추정
  - 1단계: $X_i = \pi_0 + \pi_1 Z_i + v_i$
  - 2단계: $Y_i = \beta_0 + \beta_1 \hat{X}_i + \epsilon_i$
- 진단 검정
  - 약한 도구변수 검정: 1단계 F-통계량 (Staiger & Stock, 1997)
  - 과대식별 검정: Sargan/Hansen J 검정
  - Hausman 검정: OLS vs 2SLS
- 회계연구에서의 IV 사례
  - 지배구조와 기업가치: 법적 기원을 IV로
  - 감사품질과 이익조정: 산업 전문성을 IV로
- R 패키지: ivreg (AER의 후속), sandwich, lmtest
- 데이터: 지배구조+기업가치 패널 데이터 (제공)
- 핵심 정리 + 실습과제 + 추천 논문

### 제8장. 고정효과와 임의효과 모형: plm과 fixest
- 학습목표: FE/RE 모형을 추정하고, Hausman 검정으로 모형을 선택할 수 있다
- 풀링 OLS(Pooled OLS)의 한계
- 고정효과(FE) 모형
  - 개체 고정효과: $Y_{it} = \alpha_i + X_{it}\beta + \epsilon_{it}$
  - within 변환과 LSDV
  - 시간 고정효과, 이원 고정효과(TWFE)
- 임의효과(RE) 모형
  - $Y_{it} = \alpha + X_{it}\beta + u_i + \epsilon_{it}$
  - GLS 추정
  - RE의 핵심 가정: $Cov(u_i, X_{it}) = 0$
- Hausman 검정
  - 귀무가설: RE가 일치추정량
  - 실전: 대부분의 회계/재무 연구에서 FE 선호
- plm vs fixest(feols)
  - plm: 교과서적 접근
  - fixest::feols(): 빠른 속도, 다중 고정효과, 클러스터 SE 내장
- 회계연구 적용: 기업 고정효과를 통제한 이익조정 분석
- R 패키지: plm, fixest, lmtest
- 데이터: 상장기업 패널 데이터 (제공)

### 제10장. 이중차분법(DID): 정책 효과와 인과추론
- 학습목표: DID의 식별 전략을 이해하고, staggered DID까지 적용할 수 있다
- DID의 기본 아이디어
  - 처리군(treatment) vs 통제군(control)
  - 처리 전후(before/after) 비교
  - $\hat{\delta}^{DID} = (\bar{Y}_{T,post} - \bar{Y}_{T,pre}) - (\bar{Y}_{C,post} - \bar{Y}_{C,pre})$
- 회귀 기반 DID
  - $Y_{it} = \alpha + \beta_1 Treat_i + \beta_2 Post_t + \delta (Treat_i \times Post_t) + \epsilon_{it}$
  - TWFE(Two-Way Fixed Effects) 추정
- 평행추세 가정
  - 사전 추세 검정: 이벤트 스터디 그래프
  - 위반 시 대안: 매칭 DID, 합성통제법
- Staggered DID
  - 처리 시점이 다를 때의 문제: Goodman-Bacon (2021)
  - Callaway & Sant'Anna (2021): group-time ATT
  - Sun & Abraham (2021): interaction-weighted estimator
  - fixest::sunab()
- 회계연구 적용: IFRS 의무 도입의 효과, 새로운 감사 규제의 영향
- R 패키지: fixest, did, bacondecomp, ggfixest
- 데이터: 정책 변경 전후 기업 패널 (제공)
- 핵심 정리 + 실습과제 + 추천 논문

### Part 4: 특수 연구 설계

### 제11장. 이벤트 스터디: 시장 반응 측정
- 학습목표: 이벤트 스터디 방법론을 이해하고, CAR/BHAR을 계산할 수 있다
- 이벤트 스터디의 기본 구조
  - 이벤트 정의, 이벤트 윈도우, 추정 윈도우
  - 시장모형: $R_{it} = \alpha_i + \beta_i R_{mt} + \epsilon_{it}$
- 비정상수익률(AR) 계산
  - 시장모형 AR: $AR_{it} = R_{it} - (\hat{\alpha}_i + \hat{\beta}_i R_{mt})$
  - 시장조정모형, 평균조정모형
- 누적비정상수익률(CAR)
  - $CAR_i(t_1, t_2) = \sum_{t=t_1}^{t_2} AR_{it}$
  - 검정통계량: Patell (1976), BMP (Boehmer et al., 1991)
- 장기 이벤트 스터디
  - BHAR (Buy-and-Hold Abnormal Returns)
  - 캘린더 타임 포트폴리오 방법 (CTIME)
  - 장기 이벤트 연구의 한계와 주의점
- 회계연구 적용: 공시 효과 (이익 공시, 감사의견, M&A 발표)
- R 패키지: eventstudies, tidyquant, PerformanceAnalytics
- 데이터: 주가 수익률 + 이벤트 날짜 (제공)
- 핵심 정리 + 실습과제 + 추천 논문

### 제12장. 회귀불연속설계(RDD): 경계에서의 인과효과
- 학습목표: RDD의 식별 전략을 이해하고, rdrobust로 처리효과를 추정할 수 있다
- RDD의 기본 아이디어
  - 배정변수(running variable)와 기준점(cutoff)
  - 기준점 근방에서의 무작위 배정과 유사한 상황
- Sharp RDD
  - $\tau_{SRD} = \lim_{x \downarrow c} E[Y|X=x] - \lim_{x \uparrow c} E[Y|X=x]$
  - 국소 다항 회귀(local polynomial regression)
  - 대역폭 선택: MSE-optimal bandwidth
- Fuzzy RDD
  - 기준점에서 처리 확률이 불연속적으로 변하지만 100%는 아닌 경우
  - IV 해석: Wald 추정량
- 진단과 검증
  - McCrary 밀도 검정: 배정변수의 조작 여부
  - 공변량 연속성 검정
  - 대역폭 민감도 분석
- 회계연구 적용: 지배구조 지수 기준점, 신용등급 경계
- R 패키지: rdrobust, rddensity, rdlocrand
- 데이터: 기준점이 있는 규제 데이터 (제공)
- 핵심 정리 + 실습과제 + 추천 논문

### 제13장. 분위수 회귀와 비선형 모형: 조건부 분포 분석
- 학습목표: 분위수 회귀를 수행하고, 비선형 효과를 모형화할 수 있다
- 분위수 회귀(Quantile Regression)
  - OLS가 조건부 평균만 추정하는 한계
  - $\min_{\beta} \sum_{i} \rho_\tau (Y_i - X_i\beta)$, 여기서 $\rho_\tau$는 check function
  - 분위수별 계수 비교: 상위/하위 기업에서 효과가 다른가
- 비선형 모형
  - 다항 회귀: 이차항, 교차항
  - 로지스틱/프로빗: 이항 종속변수 (부도 여부, 감사의견 등)
  - 토빗 모형: 절단된 종속변수 (배당성향, R&D 투자)
  - 포아송/음이항: 카운트 데이터 (특허 수, 소송 건수)
- 한계효과(marginal effects) 계산과 해석
- 회계연구 적용: 이익분포의 분위수별 가치관련성
- R 패키지: quantreg, margins, MASS, pscl, survival
- 데이터: 상장기업 재무데이터 (제공)
- 핵심 정리 + 실습과제 + 추천 논문

6. 심사위원 팁: "논문 심사에서 자주 지적되는 사항" 박스로 실전 팁을 제공합니다
7. 핵심 정리: 해당 장의 핵심 내용을 5~7줄로 정리합니다

### 제4장 세부 가이드: OLS 회귀분석

이분산 강건 표준오차의 중요성을 Petersen (2009)과 연결하여 설명합니다.
sandwich 패키지의 vcovHC() 사용법을 코드로 보여주되, HC0~HC4의 차이를 직관적으로 설명합니다.
회계연구에서 가장 많이 쓰이는 가치관련성 모형(Ohlson, 1995)을 예시로 활용합니다.
stargazer/modelsummary로 결과표를 작성하는 법을 이 장에서 처음 소개합니다.

### 제5장 세부 가이드: 내생성과 도구변수

내생성을 "회계연구의 아킬레스건"으로 소개하며, 심사위원이 가장 많이 제기하는 이슈임을 강조합니다.
좋은 IV를 찾기 어렵다는 점을 솔직하게 인정하고, 회계/재무 분야에서 자주 쓰이는 IV의 예시와 한계를 정리합니다.
약한 도구변수 문제와 1단계 F-통계량 > 10 규칙(Staiger & Stock, 1997)을 설명합니다.

### 제8장 세부 가이드: 고정효과와 임의효과 모형

plm으로 교과서적 추정을 보여준 뒤, fixest::feols()의 속도와 편의성을 비교합니다.
Hausman 검정의 결과가 FE를 지지하는 경우가 대부분이므로, 회계/재무 연구에서는 FE가 사실상 기본임을 설명합니다.
TWFE(이원 고정효과)의 설정과 해석을 상세히 다룹니다.

### 제9장 세부 가이드: 클러스터 표준오차

Petersen (2009, RFS)의 시뮬레이션 결과를 표와 그래프로 요약합니다.
"firm 클러스터만으로 충분한가, 이중 클러스터가 필요한가"에 대한 의사결정 기준을 제시합니다.
fixest::feols()에서 cluster = ~firm + year 한 줄로 이중 클러스터를 적용하는 방법을 보여줍니다.
Fama-MacBeth 방법을 대안으로 소개하고, 언제 각 방법이 적합한지 비교표를 제공합니다.

### 제10장 세부 가이드: 이중차분법(DID)

기본 2×2 DID부터 시작하여 TWFE, 그리고 staggered DID까지 점진적으로 확장합니다.
Goodman-Bacon (2021) 분해를 bacondecomp로 시각화하여, TWFE의 문제를 직관적으로 보여줍니다.
did 패키지(Callaway & Sant'Anna)와 fixest::sunab()(Sun & Abraham)을 모두 소개합니다.
평행추세 가정의 사전 검정을 이벤트 스터디 그래프로 시각적으로 확인하는 방법을 강조합니다.

### 제11장 세부 가이드: 이벤트 스터디

추정 윈도우와 이벤트 윈도우의 설정 기준을 실제 논문 사례로 설명합니다.
Patell t-test와 BMP t-test의 차이를 설명하고, 횡단면 상관이 있을 때 BMP를 권장합니다.
장기 이벤트 스터디의 방법론적 한계(Barber & Lyon, 1997; Lyon et al., 1999)를 솔직하게 논의합니다.

### 제14장 세부 가이드: 결과 보고와 표 작성

modelsummary를 주력으로 추천하되, stargazer와 etable도 함께 비교합니다.
한국회계학회지, The Accounting Review 등 주요 저널의 표 형식을 참고하여 템플릿을 제공합니다.
"표 하나를 10번 고치는 것보다, 코드로 자동 생성하는 것이 재현성에 핵심"이라는 메시지를 전달합니다.

### 제16장 세부 가이드: 재현 가능한 연구

targets 패키지로 분석 파이프라인을 DAG(방향 비순환 그래프)로 관리하는 방법을 소개합니다.
실제 논문 프로젝트의 폴더 구조 예시를 제공합니다.
GitHub에 코드를 공개할 때의 체크리스트(데이터 제거, README, LICENSE)를 제공합니다.
