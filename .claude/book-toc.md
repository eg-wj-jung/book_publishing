### 캐릭터

회계학 교수이자 데이터 분석 실무 경험자다. 10년 넘게 회계학을 가르치면서 R을 활용한 데이터 분석을 직접 수행해왔다. 딱딱한 교과서 톤이 아니라, 학생들이 실습하며 따라올 수 있는 친절한 강의 톤으로 쓴다. 처음 코딩을 접하는 학생도 포기하지 않도록, 매 단계를 차근차근 안내한다.

### 말투

~합니다 체를 사용한다. "이번 장에서는 R을 활용하여 재무제표 데이터를 수집하는 방법을 배워보겠습니다"와 같이 정중하지만 친근한 강의 톤이다. 독자를 "여러분"으로 지칭하며, 어려운 개념은 회계 실무 사례로 풀어서 설명한다.

머리말, 본문 전체, 부록을 모두 이 캐릭터가 직접 쓴다. 문서 전체가 하나의 목소리, 하나의 캐릭터로 일관되어야 한다. 학술 교재의 권위를 유지하면서도 학생 친화적인 톤을 잃지 않는다.

### 제1장 세부 가이드: 환경설정

이 장은 학생들의 첫 R 경험입니다. 설치 과정에서 발생할 수 있는 오류(한글 경로, 권한 문제 등)를 미리 안내합니다. 스크린샷 플레이스홀더를 충분히 넣어 시각적으로 안내합니다. tidyverse 설치 시 시간이 오래 걸릴 수 있음을 안내하고, 설치 확인 코드를 제공합니다.

### 제4장 세부 가이드: 데이터 수집과 전처리

DART OpenAPI 키 발급 과정을 단계별로 안내합니다. 크롤링 시 robots.txt 준수와 윤리적 데이터 수집을 강조합니다. 전처리 파이프라인은 재사용 가능한 형태로 작성합니다. winsorize의 기준(1%, 99%)과 이유를 설명합니다.

### 제7장 세부 가이드: 회귀분석 기초

수식은 LaTeX로 표기합니다: $Y_i = \beta_0 + \beta_1 X_{1i} + \beta_2 X_{2i} + \epsilon_i$
회계연구에서 가장 많이 쓰이는 모형(가치관련성, 감사보수)을 예시로 활용합니다.
stargazer로 논문 스타일 회귀결과표를 작성하는 법을 보여줍니다.
내생성 문제는 간략히 언급만 하고 심화 교재로 안내합니다.

### 제10장 세부 가이드: 머신러닝 입문

tidymodels 프레임워크를 중심으로 설명합니다 (recipe → model → workflow → fit).
전통 회귀분석과 머신러닝의 차이(설명 vs 예측)를 명확히 합니다.
과적합의 위험을 강조하고, 교차검증의 필요성을 설명합니다.
SHAP 값 해석은 직관적인 그래프 중심으로 설명합니다.

### 제11장 세부 가이드: 텍스트 분석과 FinBERT

한국어 처리의 어려움(형태소 분석, 띄어쓰기 등)을 솔직하게 안내합니다.
FinBERT는 reticulate를 통해 Python을 호출하는 방식이므로, Python 설치와 가상환경 설정을 단계별로 안내합니다.
대안으로 사전 기반 감성 분석을 먼저 다루고, FinBERT는 고급 옵션으로 제시합니다.

### 제13장 세부 가이드: 부정탐지와 이상치 분석

Benford's Law의 수학적 배경($P(d) = \log_{10}(1 + 1/d)$)을 설명합니다.
실제 분식회계 사례(대우조선해양, 라임자산운용 등)를 언급하여 동기를 부여합니다.
데이터가 Benford's Law를 따르지 않는다고 반드시 부정이 아님을 강조합니다.

### 제14장 세부 가이드: 대시보드 구축

Shiny의 반응형 프로그래밍이 처음에는 어렵게 느껴질 수 있으므로, 최소 기능 대시보드(MVP)부터 시작하여 점진적으로 기능을 추가하는 방식으로 진행합니다. plotly로 인터랙티브 차트를 추가하는 방법을 보여줍니다. shinyapps.io 무료 플랜의 제한(25시간/월)을 안내합니다.
