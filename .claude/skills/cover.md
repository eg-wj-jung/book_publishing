# 표지 디자인

전자책 표지를 기획하고 이미지로 생성한다. 메인안(A)과 대안(B) 2가지를 만든다.

## 시작 전 준비

`.claude/book-toc.md`를 읽고 다음을 파악한다:
- 기본 정보: 제목, 부제, 필명
- 페르소나: 책의 톤과 분위기
- 대상 독자: 표지가 어필해야 할 사람

## 장르별 표지 전략

| 장르 | 전략 |
|------|------|
| 비즈니스 실용서 | 깔끔한 타이포 중심, 1~2색, 강렬한 제목 |
| 자기계발 | 밝은 색상, 심플한 아이콘, 영감적 제목 |
| 소설 | 분위기 전달하는 일러스트/사진, 감성적 |
| 기술서 | 미니멀, 전문적, 주제 상징 아이콘 |

## 표지 규격

- 전자책: 1600x2560px (1:1.6 비율)
- 인쇄용 B5: 176x250mm (300dpi)
- 핵심 원칙: 썸네일(작은 크기)에서도 제목이 읽혀야 한다

## 산출물 구조

`output/cover_concept.md`에 아래 내용을 문서화한다:

1. 기본 정보 (제목, 부제, 저자, 장르, 타깃)
2. 장르 분석 (해당 장르의 표지 컨벤션)
3. 메인안 (A): 컨셉, 시각 요소, 타이포그래피, 색상 팔레트, 감정 톤
4. 대안 (B): 위와 동일 구조
5. 뒷표지 카피 (판매 목적 문구)
6. 띠지 추천사 제안
7. 경쟁 표지 분석 테이블

## 이미지 생성

- matplotlib을 사용하여 표지 이미지를 생성한다
- 생성 스크립트: `output/generate_cover.py`
- 메인안: `output/images/cover_a.png`
- 대안: `output/images/cover_b.png`
- 폰트는 `.claude/book-toc.md`의 퍼블리싱 사양에 정의된 폰트를 사용한다. 없으면 시스템 폰트로 폴백한다

## 타이포그래피 원칙

- 제목: 가장 크고 굵게. 표지 면적의 30% 이상 차지
- 부제: 제목의 절반 크기. 제목과 다른 색상 또는 weight
- 저자명: 가장 작게. 하단 배치
- 모든 텍스트는 썸네일(200px 너비)에서도 읽힐 수 있어야 한다
